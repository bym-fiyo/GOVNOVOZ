# GOVNOVOZ
Это мемный язык прграммирования на тематике говновоза. Создан мной по приколу, хотя ты можешь написать на нём что-нибудь серьёзное!
----
## А как установить-то ёпта?
1: клонируй репозиторий



2: помести скрипт installer.py или installer_gui.py (с крафическим интерфейсом) в домашнюю папку



3.1 (для installer.py) открой домашнюю папку в терминале и пропиши **python3 ./installer.py --install**



3.2: (для intaller_gui.py) просто нажми установить, выбери нужные опции (sudo не нужен!) и жди окончания установки!



4: ВСЁ! Проще простого ёпта!
