📦 Блоки и структура

погнали
Начало программы или блока. Всегда должен быть закрыт закончили

→ def main():
закончили
Конец блока. Закрывает погнали

→ (возврат отступа)

→ def имя(параметры):

высрать результат
→ return результат

📝 Ввод и вывод

высер
Вывести текст или значение в консоль (алиас: говняшка)

высер переменная
высер "Значение: " + x
→ print("Привет")
засрать
Ввести значение с клавиатуры и сохранить в переменную (алиас: говнись)

→ имя = input()
💡 Важно: засрать автоматически создаёт переменную и сохраняет в неё введённое значение.

🧮 Математические операции

плюс
Сложение (алиас: сложи_говно)
результат загрузили 5 плюс 3
→ результат = 5 + 3
минус
Вычитание (алиас: вычти_говно)
результат загрузили 5 минус 3
→ результат = 5 - 3
умножить
Умножение (алиас: умножь_говно)
результат загрузили 5 умножить 3
→ результат = 5 * 3
разделить
Деление (алиас: подели_говно)
результат загрузили 10 разделить 2
→ результат = 10 / 2
целочисленно
Целочисленное деление (алиас: подели_нацело_говно)
результат загрузили 10 целочисленно 3
→ результат = 10 // 3 (будет 3)
остаток
Остаток от деления (алиас: остаток_от_говна)
результат загрузили 10 остаток 3
→ результат = 10 % 3 (будет 1)
степень
Возведение в степень (алиас: степень_говна)
результат загрузили 2 степень 3
→ результат = 2 ** 3 (будет 8)
