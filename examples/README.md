# 📁 Примеры программ на Govnovoz

Это папка с примерами программ на языке **Govnovoz** — эзотерическом языке программирования с говняным уклоном.

## 🚀 Как запустить примеры

\`\`\`bash
# Установи Govnovoz (если ещё не)
python3 installer.py --install

# Запусти любой пример
govnovoz 01_hello.govno
govnovoz 02_calculator.govno
govnovoz 03_ifelse.govno
# ... и так далее
\`\`\`

## 🛠 Требования

- Python 3.6 или выше
- Установленный Govnovoz (\`python3 installer.py --install\`)

## 📝 Заметки

- Во всех примерах используется **новый синтаксис** (кроме \`13_old_style.govno\`)
- Пример \`12_navoz_example.navoz\` показывает, что \`.navoz\` работает так же, как \`.govno\`
- Для запуска примеров с файловым вводом/выводом нужны права на запись в текущую папку

## 🤝 Как добавить свой пример

1. Создай файл с расширением \`.govno\` или \`.navoz\`
2. Добавь комментарий с описанием в начале
3. Протестируй: \`govnovoz твой_файл.govno\`
4. Сделай Pull Request в репозиторий

Принимаются любые примеры — от простых до безумных! 💩
